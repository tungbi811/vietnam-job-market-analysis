# Job Market Data Pipeline

A mini data engineering project that builds an end-to-end ETL pipeline to clean,
standardize, and analyze job posting data in Vietnam.

This project focuses on data cleaning, transformation, testing, and basic analysis,
following real-world data engineering best practices.

---

## Project Objectives

* Normalize and clean job posting data
* Standardize salary information (VND / USD)
* Parse address data into city and district
* Normalize job titles into common groups
* Build a simple ETL pipeline (Extract → Transform → Load)
* Implement error handling and unit tests
* Perform basic exploratory data analysis (EDA)

---

## Results & Insights

Based on the exploratory data analysis and visualizations:

* **AI Engineer** roles have the **highest average salary** among all positions.
* **Hà Nội** has the **largest number of job postings**, followed by **Hồ Chí Minh City**.
* **Software Engineer** is the **most in-demand position**, accounting for the highest number of job listings.

These results highlight the strong demand for software engineering roles and the premium salary associated with AI-related positions in Vietnam's job market.
